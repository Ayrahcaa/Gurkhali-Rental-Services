# hamro-llc-backend

Add `.env` file, an example file `.env.example` is available to `copy & paste`.
For the mongo `DB_URI` field, include `srv` connection string.

### Build

```
$ pnpm run init
$ pnpm run build
```

### Run:dev

```
$ pnpm run init
$ pnpm run start:dev
```

### Run:prod

```
$ pnpm run start:prod
```
